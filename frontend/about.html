Developed by: Thibault Robin; University of Geneva and SIB-Swiss Institute of Bioinformatics

  • Frontend built in HTML/CSS and JavaScript with the jQuery and jQuery UI libraries
  • Parsing of plain text table files (CSV, TSV and TXT formats) powered by Papa Parse
  • Parsing of Excel files (XLS and XLSX formats) powered by SheetJS js-xlsx
  • Header pattern made with trianglify.io
  • Background tile made by Shiva from toptal.com
  • File icons made by Smashicons from flaticon.com
